# FRESHyo

Grocery price-comparison app for Roxas City, Capiz. Expo (expo-router) + Supabase.

Run: `npm start` (Expo Go supplies its own Google Maps key for dev; standalone Android builds need a real key in `app.json`'s `android.config.googleMaps.apiKey`, replacing the placeholder).

Database schema: `supabase/schema.sql` (paste into the Supabase dashboard SQL editor). Sample data: `supabase/seed.sql`.
